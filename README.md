# 📌 **Smart Vending Machine System**


## 📘 **프로젝트 개요 (Overview)**

라즈베리파이 기반으로 구현한 스마트 자판기 시스템입니다.
로컬 환경에서 동작하는 C 기반 HTTP 서버와 MySQL 데이터베이스,
그리고 메뉴 조회 / 주문 / 주문 내역 조회 / 결제 API를 제공하는 구조로 구현되어 있습니다.

본 프로젝트는 키오스크, 무인 자판기, 카페 주문기 등 다양한 IoT 디바이스에 확장 가능한 형태로 설계하였습니다.
---

## 🎯 **프로젝트 목표**

- Raspberry Pi와 STM32F411RE 간 안정적·양방향 통신 구조(UART 기반 명령–응답 프로토콜) 구축
- 서버·DB·MCU 제어 로직을 하나로 통합하는 IoT 자판기 전체 시스템 아키텍처 구현
- 통신 결과를 DB와 웹 UI로 연결하여 사용자에게 실시간 주문·배출 상태 제공
- 실제 제품화 가능한 임베디드 통신·제어 중심의 스마트 자판기 플랫폼을 구축하는 것이 최종 목표

---

## ⚙️ **주요 기능**

### 🔹 메뉴 관리 기능
- 메뉴 목록 조회
- 메뉴 가격, 카테고리 관리
- 웹 UI에서 사용자에게 메뉴 표시
- DB 변경 시 자동 반영

### 🔹 주문 처리 기능
- 주문 생성
- 주문 수량, 메뉴 ID 저장
- 주문 상태
- 주문 시간 기록
- 주문 성공/실패 대응

### 🔹 슬롯 배출  제어 기능
- IR/광센서 신호를 이용한 아이템 낙하 감지
- 센서값 전달: MCU → Raspberry Pi
- 재고 부족/걸림 문제 감지

 ### 🔹 웹 기반 UI 기능
- 메뉴, 주문, 결제 화면
- 주문 처리 결과 표시

## 🧠 ** 주요 구성 요**


| 구성 요소              | 역할 / 설명                                          |
|-----------------------|------------------------------------------------------|
| **Raspberry Pi**      | IoT 게이트웨이, HTTP 서버, DB 클라이언트             |
| **STM32F411RE 보드**  | 실시간 자판기 제어기 (MCU)                           |
| **모터 / 솔레노이드** | 각 슬롯의 아이템 배출을 위한 액추에이터             |
| **광센서 / IR 센서 (선택)** | 아이템 낙하 감지 또는 재고 / 장애 감지용 센서    |
| **버튼**              | 수동 제어나 테스트, 리셋용                           |
| **LED / 버저**        | 상태 표시 (대기, 배출 중, 오류 등) 및 알림          |
| **전원 공급 장치**     | 로직부와 모터/액추에이터용 전원 분리 구성            |

 

## 🚀 **기대 효과**

- 안정적인 서버–MCU 통신 기반의 신뢰도 높은 자판기 동작 구현
- 실시간 모터·센서 제어를 통한 배출 정확도 향상
- REST API·DB·하드웨어 통합으로 IoT 시스템 설계 역량 극대화
- 실제 물리 장비를 제어하는 경험을 통해 임베디드 실무 능력 강화
